Régression Linéaire

Fonctionnement

La régression linéaire est une méthode statistique qui permet de modéliser la relation linéaire entre une variable dépendante et une ou plusieurs variables indépendantes. L’objectif est de trouver la droite qui minimise l’erreur entre les valeurs prédites et les observations réelles.

Ce modèle est particulièrement utile pour effectuer des prédictions sur des données continues, comme par exemple estimer le prix d’un bien immobilier en fonction de sa surface.

Théorie Mathématique

La relation de la régression linéaire simple se définit par l’équation suivante :

y = β0 + β1x

Où :

  • y représente la variable à prédire,
  • x est la variable explicative,
  • β0 est l’ordonnée à l’origine,
  • β1 est le coefficient de régression qui représente l’impact de x sur y.

Pour trouver les valeurs optimales de β0 et β1, on utilise la méthode des moindres carrés, qui minimise la somme des carrés des différences entre les valeurs observées et les valeurs prédites :

Minimize Σ (yi - (β0 + β1xi))²

Exemple de Code en Python

Voici un exemple de code utilisant scikit-learn pour entraîner un modèle de régression linéaire et visualiser le résultat. 


      import numpy as np
      import matplotlib.pyplot as plt
      from sklearn.linear_model import LinearRegression
      
      # Données d'exemple
      X = np.array([1, 2, 3, 4, 5]).reshape(-1, 1)
      y = np.array([3, 4, 2, 5, 6])
      
      # Création et entraînement du modèle
      modele = LinearRegression()
      modele.fit(X, y)
      
      # Prédiction sur un intervalle
      X_test = np.linspace(0, 6, 100).reshape(-1, 1)
      y_pred = modele.predict(X_test)
      
      # Visualisation des résultats
      plt.scatter(X, y, color='blue', label='Données réelles')
      plt.plot(X_test, y_pred, color='red', label='Régression linéaire')
      plt.xlabel('Variable X')
      plt.ylabel('Variable Y')
      plt.title('Exemple de Régression Linéaire')
      plt.legend()
      plt.show()
